GIF slots:
- gallery.gif  → shown on gallery.html (Our Memories page)
- final.gif    → shown on final.html (the last page)

To use them: drop your .gif files in this folder with those names, then
follow the "REPLACE ABOVE WITH" comment right above each gif placeholder
in gallery.html / final.html to swap the placeholder div for a real <img>.
